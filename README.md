# Проект: Cuber
Цель проекта: Студенческий проект Cuber реализован в рамках программы Веб-Технологий
## Технологии, которые использовались
### Figma
  1. Построение Сетки
  2. Создание Стилей
  3. Редактирование шрифтов
  4. Создание UI kit
### HTML
  1. SEO
  2. Семантическая верстка
### CSS
  1. Методология БЭМ
  2. Флексбокс верстка
  3. Адаптивная верстка
  4. Грид верстка
  5. Медиазапросы
